# 🌐 Website Clones Collection

This repository contains a collection of **6 website UI clones** built using **HTML and CSS**.  
These projects were created to practice and strengthen frontend development skills by replicating real-world website designs.

---

## 🚀 Projects Included

- **MakerDAO Clone**
- **Kiwikisaan Clone**
- **PolicyBazaar Clone**
- **CodePen Clone**
- **Calendly Clone**
- **Flipkart Clone**

Each clone focuses on recreating the layout, design, and structure of the original website as closely as possible.

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- Flexbox  
- CSS Grid  

---


---

## ✨ Features

- Clean and well-structured code  
- Pixel-approximate UI design  
- Responsive layouts (where implemented)  
- Practice of real-world website structures

## 🎯 Purpose

- Improve HTML & CSS skills  
- Understand real-world UI/UX design
